# Pandoos Android 🐼

**Pandoos** is the official Android app for the Pandoos music ecosystem. 
*Tagline:* Life is short, relax like a Panda and enjoy music.

## 🎵 About
This repository contains the Android application for the Pandoos ecosystem, providing an emotional, animated, and seamless music listening experience. 

- **Platform:** Android (Kotlin, Jetpack Compose, Media3)
- **Base:** Forked from `maxrave-dev/SimpMusic` (GPL-3.0)
- **License:** GPL-3.0 
- **Ecosystem:** Connects to the unified Pandoos Supabase project, syncing seamlessly with our Web & Desktop clients.

## 🚀 Features (Coming Soon)
- **Cross-Platform Continue Listening:** seamlessly transition between Web, Desktop, and Android.
- **Panda Moods / Emotions:** visually expressive UI tied to your current mood.
- **Panda Now Playing:** dynamic player with animated mascot and vibe checks.
- **Smart Queue:** mood-aware and energy-level preserving playback queues.
- **Pandoos Wrapped:** detailed cross-platform listening statistics.

## 🛠️ Development Setup
To build the project locally, please refer to our internal guidelines and architecture strategy. 
1. Ensure your `local.properties` contains the required Supabase keys (`SUPABASE_URL`, `SUPABASE_ANON_KEY`).
2. Sync the project with Gradle in Android Studio.
3. Build and run the `androidApp` configuration.
